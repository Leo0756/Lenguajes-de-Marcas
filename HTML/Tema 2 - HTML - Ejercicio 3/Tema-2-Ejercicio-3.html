Tipos de textos

Los documentos de texto vienen en dos sabores: Texto rico y texto plano. El texto sin formato, como habrás adivinado, es bastante simple. Es compatible con estándar ASCII caracteres, incluidos números, símbolos y espacios, pero no admite ningún tipo de formato de texto. Por lo tanto, no puede aplicar estilos en negrita, cursiva o subrayados, y no puede usar diferentes fuentes o tamaños de fuente en un documento de texto sin formato.

Definicion

La sintaxis de un lenguaje de programación es la estructura de cómo están organizados las palabras claves, identificadores, operadores, espacios en blanco, delimitadores, corchetes, comentarios y expresiones (se denominan como sentencias), siguiendo unas reglas específicas al escribir las líneas de código para la ejecución de un programa y que son diferentes para cada lenguaje de programación.
